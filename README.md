# 拉麵分類器 MVP

這是一個以 **Next.js + React + TypeScript** 製作的逐步卡片式拉麵分類器。  
它不是單純以食材先分流，而是先從整體口味偏好切入，再逐步分析你的拉麵口味風格，最後輸出主型、相近型、食材方向與最接近的實際風格。

---

## 功能特色

- 逐步卡片式問卷流程
- 主型四軸分析
- 風味傾向判定
- 食材接受度分析
- 麵條與配料細節調整
- 過敏原排除與警示
- 正式結果頁：
  - 主型碼
  - 主型名稱
  - 四軸分析
  - 相近型提示
  - 食材方向標識
  - 最接近的實際風格
  - 原因與推薦比例

## 專案結構

- `app/`：Next.js app router 頁面
- `components/`：UI 元件與主容器
- `data/`：題庫、分類分支與原型資料
- `lib/`：reducer、推定邏輯、計分邏輯
- `types/`：型別定義
